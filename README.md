# MOYA

## Overview
MOYA is a mobile application designed to support the psychological well-being of university students through accessible, calming, and user-friendly digital tools.

The project aims to provide students with a supportive digital environment where they can access features that encourage relaxation, self-awareness, emotional balance, and healthier daily routines.

## Project Background
MOYA was developed as a project focused on student well-being and digital support. The application idea is based on the need for accessible and supportive tools that can help university students cope with stress, anxiety, and daily challenges in a more structured and comfortable way. The project includes different modules such as exercises, music, meditation, and calendar-based planning to create a more balanced user experience.

## Features
- Home page for central navigation
- Calendar page for planning and structured follow-up
- Exercise page for supportive activities
- Music page for relaxing audio content
- Meditation page for calming and focus-oriented support
- Firebase integration for content and data connection
- Simple and visually calming interface design

## Project Goal
The goal of MOYA is to provide university students with a supportive mobile environment that helps them manage stress, improve self-awareness, and interact with digital well-being tools in an accessible way. The project combines a calm user experience with practical features that can be used in daily life. The project concept includes modules such as breathing-related support, music, motivational content, and a user-friendly mobile structure built with Flutter and Firebase.

## Design Approach
The application was designed with a simple, modern, and calming visual approach. The main objective was to create an interface that feels clear, supportive, and easy to use, especially for students who may already be under stress or emotional pressure.
